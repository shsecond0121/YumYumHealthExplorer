# 🍙 냠냠건강탐험대 — 플레이 가이드

서울여자대학교 바롬설계프로젝트 4조 **'올바롬'**이 만든 식품 건강 교육 러닝 게임에 오신 걸 환영합니다!

학생 캐릭터가 학교 → 길거리 → 놀이터 → 집까지 달리며 건강한 음식은 먹고 나쁜 음식은 피하는 사이드스크롤 러너입니다.

---

## 🚀 시작하기

1. `냠냠건강탐험대.exe` 더블클릭
2. Windows에서 **"PC 보호" 경고**가 뜨면 **"추가 정보" → "실행"** 클릭 (미서명 앱이라 정상)
3. 메뉴 화면에서 **SPACE** 또는 **ENTER**로 시작!

---

## 🎮 조작법

| 키 | 동작 |
|---|---|
| **SPACE / ↑** | 점프 (공중에서 한 번 더 누르면 더블점프) |
| **↓ / S** | 슬라이드 (누르고 있는 동안 유지) |
| **Z** | 스킬 발동 (냠냠게이지 가득 찼을 때) |
| **P** | 일시정지 |
| **ESC** | 메뉴 / 이전 화면 |
| **마우스 휠 / W·S** | 도감·업적 화면 스크롤 |

💡 **공중 슬라이드 버퍼**: 점프 중에 ↓를 누르고 있으면 착지하는 순간 자동으로 슬라이드 시작!

---

## 📊 HUD 읽는 법

**좌상단**: 점수 / 최고기록 / 달린 거리(m)
**우상단**: 속도 / HP(하트 3개) / 물게이지 (가로 바)
**우상단 추가**: 자석 활성 시 남은 초, 방어막 활성 시 표시
**중앙 상단**: 10콤보부터 콤보 수 + 배율 표시 (×1 ~ ×4)
**하단 중앙**: 냠냠게이지 (가득 차면 Z 발동!)

---

## 🍽️ 음식 시스템 — 총 25종

각 스테이지마다 다른 6종(건강 3 + 정크 3) + 모든 곳 생수 1종.

### 🏫 학교
- **건강**: 우유 / 사과 / 김밥
- **정크**: 컵라면 / 초콜릿 / 사탕

### 🏪 길거리
- **건강**: 바나나우유 / 삼각김밥 / 두유
- **정크**: 떡볶이 / 햄버거 / 슬러시

### 🎠 놀이터
- **건강**: 방울토마토 / 과일주스 / 군고구마
- **정크**: 콜라 / 솜사탕 / 아이스크림

### 🏠 집
- **건강**: 쌀밥 / 미역국 / 생선구이
- **정크**: 인스턴트라면 / 배달피자 / 봉지과자

### 💧 어디서나
- **무속성**: 생수 (물게이지 +20 회복)

> **정크 음식도 점수·콤보는 들어옵니다** — 다만 5~10초 디버프(점프력 ↓, 시야 흐림, 더블점프 봉인 등)가 걸려요. **자석은 정크를 끌어당기지 않으니** 안심하고 자석 받으세요!

자세한 효과는 게임 내 **도감**(우상단 책 아이콘)에서 호버하면 모두 표시됩니다.

---

## 💧 물게이지 (갈증 시스템)

- 시작 **100**, 1m마다 -0.20씩 감소 → 약 500m에 바닥
- 생수 마시면 **+20 회복**
- 30% 이하면 주황색 경고
- **0% = 갈증**: 점프력 −20% + 콤보 누적 정지 + 빨강 깜빡임 ⚠️

생수는 약 80m 간격으로 별도 등장합니다. 적극적으로 챙기세요!

---

## 🔥 콤보 시스템

음식을 연속으로 먹을수록 점수 배율 상승:
| 콤보 | 배율 |
|---|---|
| 1~9 | ×1 |
| 10~24 | ×2 |
| 25~49 | ×3 |
| **50 이상** | **×4** |

콤보가 끊기는 조건: 일정 시간(약 1.5초) 음식 못 먹음 / 갈증 상태 / 일부 부정 효과(콤보 끊김).

---

## ⚡ 스킬 — 4종 (랜덤 발동)

냠냠게이지 가득 차면 **Z키**로 발동. 5초간 4종 중 1개가 랜덤으로 적용:

| 스킬 | 효과 |
|---|---|
| 🛡️ **무적 질주!** | 장애물 무시 |
| ⏱️ **시간 감속!** | 모든 속도 0.4배 |
| 🎉 **젤리 파티!** | 음식 자동 생성 |
| 💥 **점수 폭발!** | 점수 5배 |

게이지는 음식 종류별로 다르게 충전됩니다. 스킬 활성 중에도 게이지 충전은 계속!

---

## 💎 파워업 아이템 (날아오는 다이아몬드)

| 아이템 | 효과 |
|---|---|
| **M (자석)** | 10초간 건강·무속성 음식 자동 흡수 |
| **S (방어막)** | 다음 장애물 충돌 1회 무효 (2초 무적도 부여) |

음식 효과 중 **바나나우유**도 자석을 5초간 부여합니다 (음식 자석은 짧은 버전).

---

## ❤️ 하트 (HP 회복)

500m마다 1개씩 등장. 먹으면 HP +1 (최대 제한 없음).
HP 0이면 게임 오버.

## 📕 음식 도감

먹어본 음식만 **컬러**로 표시. 안 먹어본 음식은 **흑백 + ???**.
마우스 호버 시 툴팁(이름·속성·스테이지·효과·점수)이 떠요.

도감은 영구 누적 — 게임을 새로 시작해도 한 번 먹은 음식은 컬러로 유지됩니다.

---

## 🏅 업적 — 50개

10개 카테고리:
- **A. 진행·거리**: 100m ~ 10,000m + 첫 출석
- **B. 스테이지**: 4스테이지 도달 + 학교 귀환 + 사방 탐험
- **C. 콤보**: 10/30/50/100 콤보 + 4배율 달성
- **D. 음식 수집**: 50개/100개/긍정 30개
- **E. 음식 카테고리**: 4스테이지 마스터 + 12종 완식 + 채소·단백질
- **F. 회피·챌린지**: 깨끗한 식단 / 노콜라 / 정크 회피
- **G. 물게이지**: 첫 보충 / 5개 / 10개 / 50% 유지 / 갈증 극복
- **H. HP·생존**: 방어막 / 만피 / HP1 생존 / 하트 3개
- **I. 스킬**: 첫 스킬 / 5번 / 무적 돌파 / 4종 모두
- **J. 점수**: 5천/2만/5만 + 신기록

업적 달성 시 화면 위에서 **칭찬 스티커 팝업**이 슬라이드 다운!

# 🥚 이스터에그

## 🎽 ① 타이틀 7번 클릭 → 캐릭터 옷 색 사이클

메뉴 화면의 **"냠냠건강탐험대"** 제목을 7번 클릭하세요. 캐릭터 교복 색이 7가지로 순환합니다:

> 남색 (기본) → 핑크 → 초록 → 빨강 → 보라 → 주황 → 청록 → (반복)

✨ "교복 색깔이 바뀌었어요!" 토스트가 뜨면 성공. 다음 게임부터 적용됩니다.
*(게임 재시작하면 기본 남색으로 복귀)*

## 🔄 ② 게임 정보 → 소희 24번 클릭 → 진행 데이터 초기화

**우하단 i 버튼 → 게임 정보 화면 → 좌상단 소희 캐릭터** 24번 클릭하면 **최고점수·도감·업적이 모두 초기화**됩니다.

✨ "데이터가 모두 초기화되었어요!" 토스트가 뜨면 성공.
*(설정 — 언어/볼륨은 유지됩니다)*

⚠️ **주의**: 한 번 초기화하면 되돌릴 수 없습니다. 신중하게!

## 😊 ③ 보너스 — 소희 표정 사이클

24번 다 안 채워도, 소희 박스를 클릭할 때마다 **4가지 표정으로 사이클**합니다:
1. **기본 소희** — 안녕? 멘트
2. **안경 소희** — 엣헴! 멘트
3. **놀란 소희** — 헉! 또 눌렀어?! 멘트
4. **울상 소희** — 으앙... 멘트

각 표정마다 호버하면 다른 멘트가 나와요!

## 💡 고득점 팁

- **공중 ↓ 누르고 있기**: 착지 즉시 슬라이드 → 빠른 회피
- **콤보 유지가 곧 점수**: 정크라도 콤보가 살아있으면 다음 음식 점수가 ×4까지!
- **자석 받기 직전엔 음식 모으기**: 자석은 10초간만 효과
- **방어막은 강한 장애물 직전에**: 1회용이니 아껴두기
- **물게이지 50% 이상 1km 유지** → '수분 균형' 업적
- **HP 1로 100m 더 달리기** → '마지막 숨' 업적

## ❓ 자주 묻는 질문

**Q. SmartScreen 경고가 뜹니다**
A. 미서명 앱이라 정상입니다. "추가 정보 → 실행" 클릭하세요.

**Q. 게임 데이터는 어디 저장되나요?**
A. `%APPDATA%\냠냠건강탐험대\save_data.json` (실행 환경에 따라 다름)

**Q. 게임 화면이 작아요**
A. 800×450 고정 해상도입니다 (디자인 의도).

**Q. 다른 컴퓨터에 옮겨도 진행 데이터가 유지되나요?**
A. 위 경로의 `save_data.json` 파일을 같이 옮기면 유지됩니다.
